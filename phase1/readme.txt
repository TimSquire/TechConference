To run the program: run the main method in the Main class.

We have included sample data for tests which is specifieced in this google sheets document:
https://docs.google.com/spreadsheets/d/1OEoy0KsoStEYwMr6JGJyPQXe6ZQaX8DPHDFgEOVMGQA/edit?usp=sharing

If you wish to reset the data back to the sample data just run the class PopulateData in the Main package
If you wish to test a empty conference with no data just uncomment the specified block in PopulateData to write empty ArrayLists to the .ser files
